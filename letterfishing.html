Magnet Letter Fishing

For this activity, you will need:

- A stick or pole

- Some string or yarn

- A strong magnet

- Fridge magnet letters

- Some tape

To do this activity, you should attach a string to the end of a stick to create a fishing pole. Then attach the strong magnet to the other end of the string. Tape will help the string stay in the place you want it. Next, toss the magnet letters on the floor and the child can use the stick to "fish" up the magnet letters using their magnet "fishing pole!" As they catch the letters, help them identify each one. You could also write the alphabet on a cookie sheet and have them attach the magnet letters to the written letters as they catch each one. This activity helps children identify and recognize letters as well as understand how magnets work!
